# AI/ML Tutorial for Beginners (Python)

Hands-on machine learning tutorial for beginners using Python and Jupyter notebooks.
Learn the fundamentals: data preparation, regression, classification, evaluation, and pipelines.

## What you'll learn
- Data loading, cleaning, and splitting
- Numpy + Pandas essentials for ML
- Regression and classification with scikit-learn
- Model evaluation (RMSE, accuracy, precision/recall, ROC-AUC)
- Feature engineering basics
- Cross-validation and pipelines (best practices)
